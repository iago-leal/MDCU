# Dados Base - MDCU Framework

## Contexto do Projeto
- **Nome:** MDCU Framework
- **Propósito:** Orquestrador metodológico para Agentes de Inteligência Artificial.
- **Ecossistema:** Agentes IA (Claude, Gemini), pacotes NPX, scripts auxiliares.

## Identidade Visual e UI
- **Padrão:** CLI de terminal (npx).

## Convenções de Código e Arquitetura
- **Stack:** Markdown (skills), Node.js (CLI).
- **Abordagem:** Agent-Native para execução, Framework Distribuído Físico via NPM para instalação.
